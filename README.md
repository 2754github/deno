# deno

## Setup

```sh
$ brew install deno
$ export LINE_JARVIS_CHANNEL_ACCESS_TOKEN=...
$ export LINE_JARVIS_CHANNEL_SECRET=...
$ export LINE_NOTIFY_ACCESS_TOKEN=...
$ export OPENAI_API_KEY=...
$ deno run --allow-env --allow-net --allow-read index.ts
$ open http://localhost:8000
```
